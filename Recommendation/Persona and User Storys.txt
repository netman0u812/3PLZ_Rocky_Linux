2) Allocation plane (net_alloc)

Deterministic allocator of scarce shared resources:

tenant loopback

tenant IP block

VLAN ID

VRF name

VXLAN ID

proxy VIPs, egress IPs

Guarantees non-collision and supports audit replay (“why did tenant X get block Y?”)

3) Implementation plane (net-imp)

Applies state to infrastructure:

loopback instantiation on nodes + primary selection

OSPF advertisement

VRF/VLAN/VXLAN provisioning across nodes

per-tenant IPsec termination config

per-VRF routing processes

service proxy instantiation
